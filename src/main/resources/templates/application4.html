вставьте сюда URL страницы, которую хотите отсканировать, и нажмите кнопку "сканировать"
Если страница отсканировалась не целиком, нажмите кнопку "сканировать 1500 строк".
Когда скан страницы загружен, то для распознания текста нажмите кнопку "распознать".
Это может занять несколько минут.

И, наконец, используйте кнопку "создать урок", чтобы разделить текст на фрагменты,
содержащие по тридцать новых слов. Каждый такой фрагмент будет снабжен
контекстным словариком из тридцати слов с переводом.
